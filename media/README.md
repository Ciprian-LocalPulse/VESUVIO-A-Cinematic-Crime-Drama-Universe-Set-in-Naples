# Media Kit

- `screenshots/` — placeholder for future production stills / trailer
  frame grabs once available.
- `wallpapers/` — placeholder for downloadable fan/promo wallpapers
  derived from the key art.
- `social/social-preview.jpg` — image used for GitHub's social preview /
  Open Graph card for this repository (set under repository Settings →
  General → Social preview).

Currently populated with the existing key art; additional assets will be
added as production materials become available.
