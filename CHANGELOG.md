# Changelog

All notable changes to this repository are documented here. This project
follows a simple, human-readable versioning scheme rather than strict
semantic versioning, appropriate for a creative pitch package.

## [1.1.0] — 2026-07-18
### Added
- Full Story Bible (`docs/story_bible/Story_Bible.md`)
- Character Bible for 7 core characters (`characters/`)
- Season 1 Episode Guide (`episodes/Season_1.md`)
- Pilot script sample + full outline (`scripts/Pilot.md`)
- Pitch Deck (`pitch/PitchDeck.md`)
- Moodbook (`docs/Moodbook.md`)
- Production Bible: camera, music, editing, lighting, VFX
  (`production/`)
- AI Usage & Transparency document (`AI_USAGE.md`)
- Business Plan (`BUSINESS.md`) and Investor info (`INVESTORS.md`)
- Festival & Awards Strategy (`FESTIVALS.md`)
- Standard repository governance docs: `CONTRIBUTING.md`,
  `CODE_OF_CONDUCT.md`, `SECURITY.md`, `ROADMAP.md`
- Press kit and media folders (`press/`, `media/`, `website/`)
- Official logo assets (`logos/`)
- GitHub Actions workflows for markdown linting and link checking

### Changed
- Clarified `LICENSE` as a custom creative license (not standard MIT) to
  avoid legal ambiguity.
- Expanded `README.md` to reflect the full repository structure.

## [1.0.0] — 2026-07-18
### Added
- Initial pitch package: `README.md`, `LICENSE`, original theme track,
  bilingual lyrics, key art, process note.
