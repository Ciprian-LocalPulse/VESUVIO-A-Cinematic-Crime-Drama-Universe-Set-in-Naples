# Contributing

This repository is a **creative pitch package** for an original television
series concept created and owned by Ciprian Stefan Plesca. It is not an
open-source software project, and code-style pull requests are not the
primary purpose of this repository.

## What We Welcome
- **Issues** flagging typos, broken links, or formatting problems in the
  documentation.
- **Professional inquiries** from producers, studios, composers,
  directors, or investors interested in developing the property — please
  open an issue or see [`INVESTORS.md`](INVESTORS.md) for contact
  guidance.

## What We Don't Accept
- Unsolicited rewrites of the story, characters, or creative direction.
  Creative ownership and final decision-making remain with the creator.
- Direct edits to copyrighted creative assets (lyrics, story bible,
  scripts) without prior written agreement — see [`LICENSE`](LICENSE).

## Process
1. Open an issue describing the correction or inquiry.
2. For factual/formatting fixes, a small pull request is welcome once
   discussed in the issue.
3. For anything involving the creative property itself, please reach out
   directly rather than submitting a PR.
